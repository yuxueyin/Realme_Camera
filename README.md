# Realme Camera Enhance

一个用于补全 / 解锁 Realme / OnePlus / OPPO 相机功能的 LSPosed 模块项目。

当前项目主要围绕 `com.oplus.camera` 做运行时 Hook、配置注入与 UI 开关管理，用于启用或修复部分机型缺失的相机功能。

> 仅供学习、研究和个人设备测试使用。不同机型、不同系统版本、不同相机版本的配置差异很大，开启功能前请自行承担风险。

## 当前功能

### 相机功能

- 理光 GR 

- 影调 

- AI 风光

## 使用前提

- 已安装 LSPosed 或兼容框架。
- 模块已在 LSPosed 中启用。
- 已勾选作用域：
  - `com.oplus.camera`
- 部分功能需要 Root 权限。
- 建议修改配置后清理相机数据并重启相机。

## 支持情况

当前主要面向 Realme / OnePlus / OPPO 

不同设备、不同系统、不同相机版本之间差异较大，如果遇到无法拍摄、卡住、黑屏、闪退，需要抓取日志单独分析。

### 3. AI 风光 2 亿像素为什么输出 50MP？

当前设备实际高像素流为 8192×6144 / 6144×8192，项目当前策略是不再强行超分到 200MP，而是稳定输出 50MP，避免 APS / native 算法崩溃。

### 4. 开关明明默认关闭，为什么安装后还是开启？

如果之前已经打开过开关，SharedPreferences 里可能保存了旧状态。可以手动关闭一次，或者清除模块应用数据后重新打开。


本项目会修改相机运行时配置，可能导致：

- 相机卡住
- 黑屏
- 无法拍摄
- 相机闪退
- 图像处理异常
- 部分模式不可用

建议每次修改后保留日志，方便回退和定位。
